# 📖 GitSearch <a name="about-project"></a>

**GitSearch** is a web application that allows users to search for github users and view their profiles and repositories. It is built with React and Redux. It uses the github API to fetch users and repositories.

![image](https://github.com/Shisui6/vesper-frontend/assets/37116322/6557e826-1ebd-463e-8094-172ebb545d8f)

### Key Features <a name="key-features"></a>

- Uses the Github API to fetch users and repositories
- Users can search for github users
- Users can view the profile of a github user
- Users can view the repositories of a github user
- Responsive design

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 git
 npm
 Your favorite text editor
```

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/Shisui6/gitsearch.git
```

### Install

Install this project with:

```sh
  cd gitsearch
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm run dev - Starts the development server
```

### Testing

To run tests, execute the following command:

```sh
  npm run test - Starts the test runner
```
